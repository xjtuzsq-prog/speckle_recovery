# 散斑恢复项目

本项目用于进行散斑成像与图像恢复的基础实验，包含：

- 仿真生成目标图像
- 随机散斑/随机图样卷积成像
- 自相关计算
- 基于频域幅值约束的相位恢复
- 实验图片数据整理与后续处理

目前仓库中的核心脚本位于 [src/speckle_reco_sim.py](src/speckle_reco_sim.py) 和 [src/speckle_reco.py](src/speckle_reco.py)，其中前者主要用于仿真验证。

## 环境依赖

本项目主要依赖以下 Python 库：

- numpy
- matplotlib
- scipy
- pillow

建议 Python 版本：

- Python 3.10+

## 安装方式

如果你准备重新创建环境，可以在项目根目录执行：

```bash
python -m venv venv
source venv/Scripts/activate
pip install numpy matplotlib scipy pillow
```

如果你已经有现成环境，也可以直接安装依赖：

```bash
pip install numpy matplotlib scipy pillow
```

## 运行方式

### 1. 运行仿真脚本

```bash
python src/speckle_reco_sim.py
```

该脚本当前会完成以下流程：

1. 生成一个十字形目标图像
2. 生成随机图样
3. 在频域中进行卷积得到散斑结果
4. 计算散斑图像的自相关
5. 截取中心有效区域
6. 执行相位恢复
7. 使用 matplotlib 显示中间结果和最终恢复结果

### 2. 查看 Notebook

如果希望交互式调试或记录实验过程，可以打开：

- [src/memory_effect_sim.ipynb](src/memory_effect_sim.ipynb)

## 仿真脚本当前流程说明

[src/speckle_reco_sim.py](src/speckle_reco_sim.py) 目前大致分为以下几个部分：

### 1. 目标图像生成

脚本首先生成一个 `1024 × 1024` 的二值图像，并在中心位置绘制十字目标。

### 2. 随机图样生成

使用随机数生成 `1024 × 1024` 的随机图样，作为散斑或随机调制输入。

### 3. 卷积成像

通过 FFT 在频域中计算卷积，得到仿真的散斑图像。

### 4. 自相关处理

对散斑图像做自相关，提取其中与目标形状相关的信息。

### 5. 相位恢复

根据截取后的中心区域频谱幅值，结合迭代约束方法恢复目标图像。

## 输出结果

脚本运行后通常会弹出多张图像窗口，包括：

- 原始目标图像
- 随机图样
- 卷积/散斑结果
- 自相关截取结果
- 原目标自相关结果
- 相位恢复结果

## 注意事项

1. 当前脚本更偏向实验性验证，代码中仍有一些可以进一步整理的部分。
2. [src/speckle_reco_sim.py](src/speckle_reco_sim.py) 中存在重复定义函数等现象，后续可以继续重构。
3. 如果运行后图像窗口没有显示，请检查：
   - matplotlib 后端是否可用
   - 是否在支持图形界面的环境中运行
   - 是否被 IDE 的绘图设置拦截
4. 仓库中的 [venv/](venv/) 一般不建议提交到 Git，可在 `.gitignore` 中忽略。
